# Vrinda Store Sales Data Analysis Dashboard

## Overview
This project is an interactive Sales Data Analysis Dashboard created in Microsoft Excel to analyze e-commerce sales performance and customer behavior. The dashboard converts raw sales data into meaningful business insights using visualization and reporting techniques.

The project helps understand:
- Sales trends
- Customer demographics
- Order performance
- Top-performing states
- Sales channels contribution

---

# Dashboard Insights

## 1. Sales vs Orders Analysis
Analyzed the relationship between total sales and number of orders to identify business performance trends across different months.

### Insights:
- Identified peak sales periods
- Compared order volume with revenue generation
- Tracked monthly business growth

---

## 2. Men vs Women Purchasing Analysis
Compared purchasing behavior based on gender.

### Insights:
- Identified which gender contributes more to sales
- Compared customer engagement and order count
- Understood customer buying patterns

---

## 3. Order Status Analysis
Analyzed order completion and delivery status.

### Insights:
- Delivered orders percentage
- Cancelled and returned orders tracking
- Overall order fulfillment performance

---

## 4. Top 5 States Analysis
Analyzed highest revenue-generating states.

### Insights:
- Identified top-performing states
- Compared regional sales performance
- Found major contribution areas for business

---

## 5. Sales Channel Performance
Compared performance of different e-commerce platforms/channels.

### Insights:
- Identified highest-performing sales channels
- Compared sales contribution across platforms
- Measured customer reach by platform
